# Speech-to-text: Python Voice Recognition System
A suite of Python scripts designed for hands-free system interaction, featuring voice-activated triggers and automated alarm scheduling.

## 🚀 Overview
This project leverages speech processing to bridge the gap between human vocal commands and system-level actions. It translates real-time audio input into executable commands, demonstrating practical use cases for accessibility and home automation.

## Core Scripts
- **Voiceactivation.py:** The main listener module that processes speech-to-text input.
- **Alarm activation.py:** An automation script that triggers time-based alerts through vocal confirmation.
- **basedmemo.py:** A utility for vocal note-taking and memory storage.

## Tech Stack
- **Language:** Python
- **Key Modules:** SpeechRecognition, PyAudio (for real-time audio capture).

## Future Enhancements
